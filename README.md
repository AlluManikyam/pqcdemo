# PQC Knowledge Portal (CRA + Tailwind) - v2

Features:
- SPA with sections (Why, What, Migration, Docs, Avatar, Contact)
- Tailwind CSS (dark mode via class)
- Manual Dark Mode toggle
- Smooth scroll navigation (react-scroll) with offset for sticky header
- Mobile hamburger drawer (visible <= 768px)
- Floating Chat button opens modal with embedded YouTube video (paused by default)

Usage:
  npm install
  npm start
